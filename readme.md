# Инструкция по запуску проекта

1. **Установка Node.js**
   - Скачайте и установите Node.js с официального сайта [Node.js](https://nodejs.org/).
   
2. **Скачивание проекта**
   - Скачайте проект с репозитория.

3. **Открытие проекта в VS Code**
   - Откройте проект в редакторе VS Code.

4. **Открытие терминала**
   - Откройте терминал в VS Code, используя сочетание клавиш  `Ctrl + ~` .

5. **Установка зависимостей**
   - В терминале выполните команду  `npm install`  для установки необходимых зависимостей.

6. **Запуск проекта**
   - После установки зависимостей выполните команду  `npm run dev`  для запуска проекта с помощью Live Server.

7. **Просмотр проекта**
   - Откройте браузер и введите адрес  `http://localhost:`  для просмотра проекта.

Приятной работы!
